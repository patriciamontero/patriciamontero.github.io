Buttons #

A button is a distinct visual element where users can click or tap to trigger an action.

Guidelines

  • Use a <button> for actions, such as copy, expand, collapse or send. If the need is a change of location, use a link <a>. The proper use of their meaning as HTML elements results in a more inclusive experience. It also eases the maintenance and scaling of this design system.
  • Deliver every button with a precise text that states its purpose. In most cases, it should be a verb that incites action. You'll know it works when the user can predict the outcome before triggering the button.
  • Keep its content below four words to prevent text wrapping.
  • Use sentence case for the texts, which means that only the first word starts with a capital letter. Don't use a period at the end of the phrase.
  • A primary button conveys the most prominent choice that the user can take in a section. For this reason, there should be only one per page. For supporting actions, pick a secondary button instead.
  • A button's colour changes depending on its surroundings. It may be any of the four shades picked from the colour palette, as long as it yields an accessible contrast. The shades are dark, accent, white and black:
  • #dd2808
  • #f96c53
  • #f8f8f8
  • #2f2f2f

Code snippets

Primary dark button

White text (#f8f8f8) over a dark box area (#dd2808), this is the default style for all buttons. It can only perform over a light-coloured background or contrasting image.

Here are a few examples of proper usage:

                                
                                    

                                        HTML
                                        
                                    

<button>This is a button</button>
                                    

                                        CSS
                                        
                                    


button {
    background-color: #dd2808;
    border: 0;
    border-radius: 0.1875rem;
    color: #f8f8f8;
    font-family: 'Raleway', sans-serif;
    font-size: 1rem;
    font-weight: 700;
    padding: 0.5rem 0.75rem;
    text-decoration: none;
}
button:hover {
    background-color: #a71e06;
}

Primary accent button

Black text (#2f2f2f) over an accent colored area (#f96c53), this is an alternative style. It can only appear over a very dark-coloured background or contrasting image.

Here are a few examples of proper usage:

                                
                                    

                                        HTML
                                        
                                    

<button class="accentbtn">This is a button</button>
                                    

                                        CSS
                                        
                                    


.accentbtn {
    background-color: #f96c53;
    border: 0;
    border-radius: 0.1875rem;
    color: #2f2f2f;
    font-family: 'Raleway', sans-serif;
    font-size: 1rem;
    font-weight: 700;
    padding: 0.5rem 0.75rem;
    text-decoration: none;
}
.accentbtn:hover {
    background-color: #fba798;
}

Secondary dark button

A white box area (#f8f8f8) with dark border and text (#dd2808), this is the main style for secondary buttons. It can work over a light coloured background or contrasting image.

Here are a few examples of proper usage:

                                
                                    

                                        HTML
                                        
                                    

<button class="secondary">This is a button</button>
                                    

                                        CSS
                                        
                                    


.secondary {
    background-color: #f8f8f8;
    border: 2px solid #dd2808;
    border-radius: 0.1875rem;
    color: #dd2808;
    font-family: 'Raleway', sans-serif;
    font-size: 1rem;
    font-weight: 700;
    padding: 0.5rem 0.75rem;
    text-decoration: none;
}
.secondary:hover, .secondary:focus:hover {
    background-color: #fcdad4;
}
