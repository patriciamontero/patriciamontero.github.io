Navigation bar#

The navigation bar—or navbar—enables shifting between the essential pages. It should be evident without taking up too much space, especially on mobile.

Guidelines

  • The navigation bar is a list that can contain links and buttons. Links appear underlined and redirect users to other pages. Buttons, instead, open another list of related links.
  • Given its fundamental browsing role, the navbar must be available at all times. One way to achieve this is by using a sticky navigation bar that stays on top of the page.
  • Whenever possible, keep under four the total items in the navbar. That reduces the need for a hamburger menu on mobile devices.
  • Identify the current page on the navbar with a thicker underline. It may allow the users to know their location on the website.
  • To label each item, use as few words as possible. It eases the creation of compact menus that are easier to read and occupy the less real state.
  • To avoid confusion, set a high contrast between the navbar and the rest of the page.
  • Use sentence case for the texts, which means that only the first word starts with a capital letter.

Code snippets

Navbar on a light background

Black text (#f8f8f8) over a light coloured area. In the hover state, the main items change to the lightest accent (#fcdad4).

Here are a few examples of proper usage:

                                        HTML
                                        
                                    


<nav>
    <ul>
        <li><a href="index.html" >Brand</a></li>
        <li><a href="styles.html">Styles</a></li>
        <li><a class="current" href="components.html">Components</a></li>
    </ul>
</nav>

                                    

                                        CSS
                                        
                                    


nav a:link, nav a:visited {
    background-color: transparent;
    color: #2f2f2f;
    font-family: 'Raleway', sans-serif;
    font-weight: 700;
    line-height: 1.2;
    padding: 0 0.25rem;
    transition: all 0.4s ease-in;
}
nav a:hover, nav a:focus, nav a:visited:hover  {
    outline: 0.25rem dotted #dd2808;
    text-decoration: none;
}
nav li {
    display: inline-block;
    padding: 0.5rem;
    margin: 0;
}
nav ul {
    display: inline;
}
a.current {
    border-bottom: 0.25rem solid #2f2f2f;
    border-radius: 0;
    cursor: default;
    text-decoration: none;
}
