Texts #

Along with illustration and design pieces, this is the core of everything I publish on the Web. It must be clear and readable at all times.

Guidelines

  • Only apply two typefaces in one weight each: Raleway (bold) and Georgia (regular). The details about their use are available in the style guide, under typography.
  • Use Raleway for headings, but set Helvetica and sans-serif as a fallback. Raleway is a Google font that you can download here.
  • Use Georgia for body text, but set Times and serif as a fallback. Georgia is a web-safe font, which means that it's available in most computers.
  • For code snippets, use the browser's default typeface.

Headings

Their styling is simple, underlined by default. Plus, a slight background colour change enforces the interaction.

Since these links are part of a paragraph, they should only appear over a plain background to remain legible.

Body text

These statements need an instant response from the user, and they usually appear at the end of a section. Their sentences end without a period.

Since these links are detached phrases, they perform well over any contrasting background.

                                    HTML
                                    
                                

<p><a class="cta" href="#0">Another link as example</a></p>
                                

                                    CSS
                                    
                                


a:link, a:visited {
    background-color: #fcdad4;
    border-radius: 3px;
    color: #2f2f2f;
    display: inline-block;
    font: 700 1.25rem/1.6 'Raleway', Helvetica, sans-serif;
    margin: 1.44rem 0 1.2rem;
    padding: .25rem .5rem;
    transition: all .4s ease-in;
}

a:hover, a:visited:hover, a:focus {
    background-color: #fba798;
    text-decoration: none;
}
