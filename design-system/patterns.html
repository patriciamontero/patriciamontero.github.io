Links #

A link is a navigational text element. It should be brief and descriptive of the intended destination.

On paragraphs

This link doesn't work.

                        

HTML

<p><a href="#">This link</a> doesn't work.</p>

CSS

a:link, a:visited {
    background-color: #fcdad4;
    color: #2f2f2f;
    display: inline-block;
    padding: 0 0.25rem;
    transition: all 0.4s ease-in;
}
a:hover, a:visited:hover, a:focus {
    background-color: #fba798;
    text-decoration: none;
}

Cards#

Cards are a compound of image (thumbnail), text (title and intro) and a button. Their main goal is to highlight key content and take the user to it.
