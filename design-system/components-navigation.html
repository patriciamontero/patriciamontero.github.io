Navigation #

A link is a navigational element. It should be brief and descriptive of the intended destination.

Guidelines

  • Use text links in favour of buttons for navigation purposes. If it takes you to a new page or section of a page, it should be an anchor element <a>.
  • Provide text rather than images, so the content reminds readable. The only exception is the use of logos, as in the case of social media.
  • Keep the content below four words to prevent text wrapping.
  • Use meaningful text that gives users an idea of where it's taking them. Avoid the use of vague sentences like Click here.
  • Set a contrast between body text and background colour that complies with standards. Anyone should be able to read it.
  • Always underline active links. Reinforce them in the hover state with a cursor pointer and hiding the underlining.
  • Set the font size in relative units to keep it proportional to the reading device.
  • The focus state must be explicit: a solid contrasting line (#49f9e2). It allows users to distinguish the content they are currently selecting.

Code snippets

Links in the main menu

These links allow browsing between the key pages, and so they should be available at all times. A thicker underline identifies the current page.

Whenever possible, keep under four the total items in the main navigation bar. It may be viable, given the simplicity of my interfaces, and it reduces the need for a hamburger menu.

                                        HTML, black text
                                        
                                    


<nav>
    <ul>
        <li><a href="index.html" >Brand</a></li>
        <li><a href="styles.html">Styles</a></li>
        <li><a class="current" href="components.html">Components</a></li>
    </ul>
</nav>

                                    

                                        CSS, black text
                                        
                                    


nav a:link, nav a:visited {
    background-color: transparent;
    color: #2f2f2f;
    font-family: 'Raleway', sans-serif;
    font-weight: 700;
    line-height: 1.2;
    padding: 0 0.25rem;
    transition: all 0.4s ease-in;
}
nav a:hover, nav a:focus, nav a:visited:hover  {
    outline: 0.25rem dotted #dd2808;
    text-decoration: none;
}
nav li {
    display: inline-block;
    padding: 0.5rem;
    margin: 0;
}
nav ul {
    display: inline;
}
a.current {
    border-bottom: 0.25rem solid #2f2f2f;
    border-radius: 0;
    cursor: default;
    text-decoration: none;
}
